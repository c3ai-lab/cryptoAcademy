# CryptoCademy

## Commands

### Run development docker container(s):

```bash
$ bash docker/developer/startup
```

ODER 

```bash
$ chmod +x docker/developer/startup
$ ./docker/developer/startup
```

### Build VueJS frontend (hot reload):

```bash
$ bash npm-exec run watch
```

ODER 

```bash
$ chmod +x npm-exec
$ ./npm-exec run watch
```
